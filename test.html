|strong| |/strong|

The strong tag marks text inside it as strong. It is not a visual hint, but a semantic hint. Depending on the medium used, what it does will vary. Browsers typically make the text bold.

|em| |/em|

The em tag is used to emphasize text, in the same way strong does. Browsers typically italicize the text.

|blockquote| |/blockquote| AND |q| |/q|

The blockquote and q tags allow quotes in text, blockquote applies quotes as a block element, q applies inline quotation. inline quotation "with regular quotation marks"
