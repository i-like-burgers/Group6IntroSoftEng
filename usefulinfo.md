# Useful Info

## Local Setup

  Run the database migration, seed the database, then start the server:

- `npm run prisma:migrate`
- `npm run prisma:seed`
- `npm start`

## Seeded Accounts

Default seeded logins:

- admin: `admin` / `admin1234`
- seller: `seller` / `seller1234`
- buyer: `buyer` / `buyer1234`
